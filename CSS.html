Tag References

Selector Name Description
.class Class A group of elements that are the same or similar. (Can contain multiple values)
#id ID Unlike classes can only have one single value.
* Universal Targets all elements

Declaration Block References

Declaraiton Name Desription
font-family: Font They style of font your selector is to be written in.
font-size: Font Size The size that you want your font written in. (Usually measured in pixedls.)
font-weight: Font Weight How bold or thin your text appears.
text-align: Text Align Aligns text on page. (Left, center, right, justify=spaces out text)
color: Foreground Color The color your text will appear in.
background-color: Background Color The color your background will appear in.
opacity: Opacity The measure of how transparent your selector will be. (Measured from 0-1, 1 representing 100% or fully visible)
background-image: Background Image Sets an image for your background.
text-transform: Text Transform Used to make text appear all uppercase or lowercase.
letter-spacing: Letter Spacing Spacing between individual letters.
word-spacing: Word Spacing Spaceing between words.
line-height: Line Height Sets how tall we want each line.
