Tag References

Tag Name Description
<head> Head The top of your HTML Doc containing your meta, CSS link, title, etc. (Always remember, I can see your body, but I can't see whats in your head)
<body> Body Part of the HTML page that contains everything the user will be able to see on the web page
<main> Main Bulk of the content on the page inluding the header(s), but not the footer.
<footer> Footer Content info, copyright info, terms of use, site map, reference to top of page links. (bottom of page in small text)
<nav> Navigate Creates navigation tabs on webpage. (always linked to another page)
<link> Link Links one HTML page to other styleing pages like CSS, Javascript, etc. (Found in <head>)
<a> Anchor Used with href to define a hyperlink, which can be used to link one page to another. (used with <nav>)
<section> Section Seperates parts of your webpage almost like chapters in a book
<article> Article Used for articles, blogs, comments, magazines, etc. (sections and articles can be put inside one another)
<aside> Aside Used to enhance another element, but isn't required to understand the main content.
<span> Span A inline container used to mark up a part of text, or a part of a document. (Easily styled by CSS or manipulated with JavaScript us the class or id attribute)
<div> Division Defines a devision or a section in a HTML document. (Easily styled by CSS or manipulated with JavaScript us the class or id attribute)
<figure> Figure Used to encapsulate media such as images, illistration, diagram, code, snippet, etc.
<figcaption> Figure Caption Creates a Caption for media found on page
